## 🎨 Image Generation

### How to Use:

1. **Click the Image Icon** 🖼️ in the input box
2. **Enter a prompt** describing the image you want
3. **Select a model** (optional):
   - Stable Diffusion 2.1 - Best quality
   - Stable Diffusion 1.5 - Fast
   - OpenJourney - Artistic style
   - Stable Diffusion 1.4 - Classic
4. **Click "Generate Image"**
5. Wait 5-15 seconds for generation
6. **Insert into chat** or save the image

### Example Prompts:
```
"A beautiful sunset over mountains, photorealistic"
"A cute robot reading a book, cartoon style"
"Futuristic city with flying cars, cyberpunk"
"A cozy coffee shop interior, warm lighting"
"Abstract art with vibrant colors"
```

### Tips:
- Be specific and descriptive
- Mention style (photorealistic, cartoon, oil painting, etc.)
- Include lighting details (warm, dramatic, soft)
- Add mood/atmosphere keywords
- First generation may take longer (model loading)

### Limits:
- **Free tier**: ~1,000 images/day
- **Generation time**: 5-15 seconds
- **Resolution**: 512x512 pixels

## 🔍 Web Search

### How to Use:

1. **Click the Search Icon** 🔍 in the input box
2. **Enter your search query**
3. **Click "Search"** or press Enter
4. Get AI-summarized results with sources
5. **Insert into chat** to discuss with AI

### What It Does:
- Searches Google for your query
- Gets top 5 results
- AI summarizes the information
- Provides clickable source links
- Cites sources in summary

### Example Searches:
```
"Latest news about AI technology"
"How to make sourdough bread"
"Best practices for React hooks"
"Current weather in New York"
"Python vs JavaScript comparison"
```

### Features:
- **AI Summary** - Concise answer from multiple sources
- **Source Citations** - [1], [2], etc. in summary
- **Clickable Links** - Open sources in new tab
- **Insert to Chat** - Discuss results with AI

### Limits:
- **Free tier**: 100 queries/day
- **Results**: Top 5 sources
- **Response time**: 2-5 seconds

## 🔧 Troubleshooting

### Image Generation Issues:

**"Model is loading, please try again"**
- Wait 3 seconds, it will auto-retry
- First generation takes longer

**"Failed to generate image"**
- Check your prompt (avoid inappropriate content)
- Try a different model
- Simplify your prompt

**Slow generation**
- Normal for first image (model loading)
- Subsequent images are faster
- Try Stable Diffusion 1.5 for speed

### Web Search Issues:

**"Search failed"**
- Check internet connection
- Verify API key in .env
- Check daily quota (100/day)

**No results**
- Try different search terms
- Be more specific
- Check spelling

**Quota exceeded**
- Free tier: 100 searches/day
- Resets at midnight UTC
- Upgrade for more quota
