# 🚀 Quick Reference Card

## Start Server
```bash
npm run dev
```
Open: `http://localhost:3000`

---

## 🎨 Image Generation
**Button**: 🖼️ icon in input box

**Quick Use**:
1. Click 🖼️
2. Type: "A cute robot"
3. Click "Generate"
4. Wait 10 seconds
5. Insert to chat

**Free Limit**: 1,000/day

---

## 🔍 Web Search
**Button**: 🔍 icon in input box

**Quick Use**:
1. Click 🔍
2. Type: "Latest AI news"
3. Press Enter
4. Get AI summary + sources
5. Insert to chat

**Free Limit**: 100/day

---

## ⌨️ Keyboard Shortcuts
- `Ctrl+K` - New chat
- `Ctrl+/` - Search
- `Ctrl+E` - Export
- `Ctrl+Shift+T` - Toggle theme
- `Esc` - Stop generation

---

## 🎯 Quick Actions

### On Messages:
- **Copy** - Copy text
- **Regenerate** - Retry response
- **Like/Dislike** - React
- **Pin** - Bookmark
- **Speak** - Text-to-speech

### On Chats:
- **Right-click** → Options menu
- **Pin** - Keep at top
- **Share** - Get link
- **Export** - Download (TXT/MD/PDF)
- **Delete** - Remove

---

## 🎤 Voice Input
**Button**: 🎤 icon in input box

1. Click microphone
2. Speak your message
3. Click again to stop
4. Text appears in input

---

## 📤 Export Chat
1. Right-click chat in history
2. Select "Export"
3. Choose format:
   - 📄 TXT
   - 📝 Markdown
   - 📕 PDF

---

## 🔒 2FA Setup
1. Settings → Two-Factor Authentication
2. Scan QR code
3. Enter 6-digit code
4. Done!

---

## 🎨 Theme Toggle
- Click sun/moon icon
- Or press `Ctrl+Shift+T`

---

## 📁 Organize Chats
- Right-click chat
- Move to folder
- Add tags
- Pin important ones

---

## 🔍 Search in Chat
- Click search icon
- Or press `Ctrl+/`
- Type to find messages

---

## 📌 Pin Messages
- Hover over message
- Click pin icon
- View pinned in header

## 🐛 Quick Fixes

**Image not generating?**
- Wait 3 seconds, auto-retries
- Try different model
- Simplify prompt

**Search not working?**
- Check internet
- Try different query
- Check daily limit

**Voice not working?**
- Use Chrome/Edge
- Allow microphone
- Check permissions
